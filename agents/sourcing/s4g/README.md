# S4G

**S4G** is a guided sourcing orchestration agent for **complex, multi-stakeholder** categories. It packages timelines, decision scenarios, and governance-ready narratives so teams move from strategy to execution without losing the plot.

## Why it matters

Big sourcing events fail on **misalignment**, not spreadsheet math. S4G keeps **lots, incumbents, constraints, and decision forums** in one coherent playbook.

## Common inputs

- Signed-off strategy (from **Source AI** or category plan)
- Supplier qualification matrix
- Baseline commercial posture and volume splits

## Typical outputs

- Playbook timeline with milestones and owners
- Scenario matrix: award patterns, transition risks, guardrails
- Decision memo draft for steering committees

## Workflow in practice

Kick off after strategy approval; feed **RFX Creator** for artifacts and **eRFX Agent** for operational event cadence.

## Extension opportunities

- Integrate **Negotiation Coach** outputs for renewal-heavy categories
- Auto-link savings baselines to **Savings Tracker** fields

## Branding note

Folder name stays `s4g`; display name **S4G** is used in metadata and documentation.
