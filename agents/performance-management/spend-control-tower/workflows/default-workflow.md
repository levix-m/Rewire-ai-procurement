# Default workflow — Spend Control Tower

## Goal

Convert raw P2P exceptions into **owned remediation** within a sprint cycle.

## Steps

1. **Snapshot** data as-of date; freeze definitions.
2. **Classify** exceptions using agreed taxonomy.
3. **Prioritize** by materiality, risk, and repeat offenders (aggregate).
4. **Root cause** workshops with P2P + category + IT.
5. **Playbook assignment** — rule, catalog, training, supplier fix.
6. **Implement** changes with change control.
7. **Measure** post-fix window; compare to baseline.
8. **Report** outcomes to finance/compliance stakeholders.

## Escalation

Suspected **fraud rings** or **repeat intentional bypass** → route to investigations, not public dashboards.
