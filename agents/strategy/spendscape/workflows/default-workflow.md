# Default workflow — Spendscape

## Goal

Produce a **versioned spend landscape** for a category or region that is credible enough to anchor a steering committee.

## Preconditions

- Spend extract refreshed within agreed SLA (state “as-of” date).
- Supplier roll-up rules available (even if imperfect).
- Category tree version noted (taxonomy releases cause shifts).

## Steps

1. **Frame** — Confirm scope: category nodes, regions, capex vs opex treatment, inclusions/exclusions.
2. **Normalize** — Validate currencies, UoM, and time window; document FX source.
3. **Baseline landscape** — Compute spend totals, concentration, fragmentation, site spread.
4. **Contract overlay** — Map coverage vs spend; highlight “uncontracted but material” bands.
5. **Opportunity thesis** — Generate 5–12 hypotheses with effort/risk/confidence; avoid duplicates.
6. **Stress-test** — Sanity check against prior quarter and known business events (plant shutdowns, etc.).
7. **Package** — Executive summary + appendix tables; include data defect register.
8. **Route** — Category director review → optional sourcing COE intake for **Source AI** handoff.

## Escalation

- **>30% uncategorized spend** → pause opportunity ranking; open data remediation ticket.
- **Single-source critical spend** → flag business continuity, not just “savings.”
