# Assets — template

## What belongs here

Supporting material that is **not** plain Markdown prompts:

- Architecture diagrams (PNG, SVG)
- Slide templates for QBRs or steering committees
- Sample CSV/JSON **schemas** (not production data)
- Links to Figma or Miro boards (describe access policy in prose)

## What does not belong here

- Secrets, tokens, or full contract PDFs with sensitive terms
- Large binaries without documentation of purpose and owner

## Naming

Use descriptive, versioned filenames: `spend-heatmap-v2.excalidraw.svg`.

Update this README when adding files so contributors know what each asset is for.
