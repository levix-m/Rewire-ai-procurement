# eRFX Agent

The **eRFX Agent** runs and **coordinates sourcing events**—the operational discipline of deadlines, Q&A fairness, submission compliance, and a clean hand-off to evaluation.

## Why it matters

Even brilliant RFx documents fail when **process breaks**: missed amendments, uneven answers, or chaotic submissions. This agent keeps the **event machine** predictable and auditable.

## Common inputs

- Event charter: due dates, lots, incumbency rules, confidentiality terms
- RFx documents ready for publication
- Supplier list with contact roles

## Typical outputs

- Runbook with weekly checkpoints and RACI
- Per-supplier compliance tracker (forms, attachments, declarations)
- Committee pack: submission index, clarifications log, integrity notes

## Workflow in practice

Use after **RFX Creator** completes drafts and **S4G** locks scope. Pair with negotiation agents during **clarification windows**.

## Extension opportunities

- Automated **submission validation** (file types, required worksheets)
- Integration with **supplier portals** for status dashboards

## Differentiation

- **RFX Creator** focuses on **authoring** content.
- **eRFX Agent** focuses on **running** the event and **operational compliance**.
