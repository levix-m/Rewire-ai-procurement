# Default workflow — RFX Creator

## Goal

Produce a **Legal/Tech-reviewed** RFx package ready for **eRFX** publication.

## Steps

1. **Scope intake** — objectives, constraints, lots, regions.
2. **Outline** — assign section owners (Legal, Security, Ops, Category).
3. **Draft v0** — requirements, instructions, schedules (non-binding phrasing where needed).
4. **Scoring model** — align to strategy; avoid double counting.
5. **Red-flag review** — category lead + sourcing excellence.
6. **Legal/Security hardening** — replace placeholders with approved clauses.
7. **Version & hash** — store v1.0 baseline for event integrity.
8. **Hand-off** — publish package to **eRFX Agent** checklist.

## Escalation

**Biased specifications** detected → stop; reformulate objective criteria.
