# Default workflow — Source AI

## Goal

Produce a **reviewed sourcing strategy** that can feed **S4G** or **eRFX Agent** runbooks without rework.

## Steps

1. **Clarify objectives** — Cost, resilience, innovation, ESG weights; non-negotiables.
2. **Baseline reality check** — Validate spend/supplier snapshot; flag data gaps affecting strategy.
3. **Market context** — Summarize relevant signals; cite sources or mark as anecdotal.
4. **Generate options** — At least two viable paths; include incumbent management approach.
5. **Select recommendation** — Default path + decision gates (e.g., if only one qualified bidder).
6. **Long list & diligence** — Objective criteria; note sanctions/diversity checks.
7. **Risk register** — Transition, quality, IP, data security as relevant.
8. **Handoff** — Structured brief for **RFX Creator** (sections) and **eRFX Agent** (milestones).

## Escalation

- **Sole-source justification** required by policy → trigger governance workflow before external invites.
