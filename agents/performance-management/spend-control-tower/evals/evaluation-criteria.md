# Evaluation criteria — Spend Control Tower

1. **Privacy respect** — Aggregates by default; careful with individual data.
2. **Materiality** — Focuses leadership attention on meaningful exposure.
3. **Actionability** — Playbooks map to owners and realistic interventions.
4. **Neutral tone** — No unfounded accusations; suggests investigations when warranted.
5. **Taxonomy discipline** — Consistent exception classes across reports.
6. **Trend accuracy** — Describes directionally correct patterns from inputs.
7. **Differentiation** — Not a category strategy doc; stays P2P control focused.
8. **Control mindset** — Never recommends circumventing approvals.
