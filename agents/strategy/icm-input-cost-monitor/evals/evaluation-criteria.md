# Evaluation criteria — ICM

1. **Feed integrity** — Uses provided index values; never invents market data.
2. **Assumption transparency** — Weights, lags, caps documented clearly.
3. **Procurement utility** — Briefing supports real conversations, not generic macro commentary.
4. **Contract humility** — Does not over-interpret clauses; escalates ambiguity.
5. **Scenario discipline** — Ranges are coherent; math directionally correct.
6. **Leverage awareness** — Combines market facts with supplier-specific mechanics.
7. **Differentiation** — Stays separate from tail spend hygiene (SpendSculptor) and pure landscape analytics (Spendscape).
8. **Risk tone** — Professional; no supplier vilification.
