# Default workflow — Category Strategy Agent

## Goal

Publish a **signed-off category strategy** artifact for a fiscal planning cycle.

## Steps

1. **Stakeholder intake** — Interview summaries encoded as constraints (no PII).
2. **Baseline performance** — Spend, supplier count, incidents, savings history.
3. **Strategic themes** — 3–5 pillars max; reject overcrowded strategies.
4. **Lever design** — Demand vs supply actions per theme.
5. **Roadmap** — Quarters with dependencies; mark quick wins vs structural bets.
6. **KPI & target hygiene** — Ensure definitions match Finance rules.
7. **Risk/ESG** — Materiality-based, not checkbox filler.
8. **Review cadence** — Monthly ops vs quarterly exec forums.
9. **Approval** — Category director + BU sponsor sign-off recorded.

## Escalation

Conflicting BU priorities → escalate to **portfolio steering**, not hidden compromise text.
