# Evaluation criteria — Supplier Performance Cockpit

1. **Signal quality** — Highlights truly unusual movements; avoids crying wolf.
2. **Clarity** — Exec headline matches operational detail.
3. **Neutrality** — Professional supplier language; no blame without evidence.
4. **Drill guidance** — Suggests dimensions that would actually explain variance.
5. **Data literacy** — Notes measurement lag/definition risks.
6. **Action boundary** — Suggests SPM hand-off when governance needed.
7. **Differentiation** — Does not draft full CAPA (SPM domain).
8. **Export usefulness** — BI export guidance is concrete.
