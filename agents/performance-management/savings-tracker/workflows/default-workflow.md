# Default workflow — Savings Tracker

## Goal

Produce a **finance-reviewable validation** for each material initiative.

## Steps

1. **Charter check** — objectives, scope, owner, effective date.
2. **Baseline lock** — documented, approved, versioned.
3. **Measurement plan** — data sources, frequency, exclusions (FX, one-offs).
4. **Realization window** — wait for stable operations post-transition.
5. **Compute** — price/volume/mix decomposition where feasible.
6. **Validate** — FP&A sign-off per policy.
7. **Leakage review** — P2P compliance, contract drift, off-contract buys.
8. **Close/update** — status change in portfolio registry.

## Escalation

Dispute on baseline → **VMO steering** with Finance chair.
