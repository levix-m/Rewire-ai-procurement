# Default workflow — Contract AI

## Goal

Produce a **reviewable clause map** for a contract or portfolio slice.

## Steps

1. **Ingest** documents with version control; note governing law if stated.
2. **Normalize** amendments (what each changes).
3. **Extract** clauses per taxonomy with citations.
4. **Cross-check** internal consistency (defined terms, exhibits).
5. **Draft obligations & renewals** as structured lists.
6. **Legal review** — attorneys confirm interpretations; adjust map.
7. **Publish** to CLM metadata and link to operational matchers (invoice compliance).
8. **Monitor** triggers for upcoming notice windows.

## Escalation

Ambiguous liability/indemnity language → **mandatory** Legal review before automation.
