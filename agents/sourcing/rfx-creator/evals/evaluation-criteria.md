# Evaluation criteria — RFX Creator

1. **Completeness** — Major RFx sections present; owners assigned.
2. **Evaluability** — Scoring model maps to measurable supplier responses.
3. **Fairness** — Flags biased specs; suggests objective alternatives.
4. **Legal humility** — Uses placeholders; no fabricated binding clauses.
5. **Clarity** — Suppliers could respond without constant clarifications (within reason).
6. **Risk coverage** — EHS, data, subcontracting, IP addressed.
7. **Differentiation** — Does not absorb event operations (eRFX domain).
8. **Procurement polish** — Professional, organized, consistent terminology.
