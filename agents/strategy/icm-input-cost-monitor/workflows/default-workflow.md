# Default workflow — ICM

## Goal

Produce a **decision-ready market shock brief** for category leadership.

## Steps

1. **Snapshot indices** — Lock feed timestamps; note vendor of record.
2. **Map exposure** — Link categories/suppliers to drivers via weights.
3. **Model scenarios** — Base/downside/upside; show math at high level.
4. **Contract cross-check** — Indexation, caps, lookback windows (user-provided summaries).
5. **Sourcing implications** — Timing for RFx, dual sourcing, hedging (if applicable).
6. **Negotiation package** — Fact sheet + questions to ask suppliers.
7. **Review** — Cost engineering validates weights; legal reviews clause interpretation claims.

## Escalation

If clause text is ambiguous → **stop** automated recommendations; route to Legal.
