# Default workflow — S4G

## Goal

Align stakeholders on a **single sourcing program model** before external communications begin.

## Steps

1. **Validate inputs** — Strategy, supplier qualification, baselines.
2. **Define lots & rules** — Incumbency, transitions, geographic splits.
3. **Build timeline** — Strategy lock → RFx release → Q&A → submissions → eval → award.
4. **Scenario planning** — 3–5 credible award patterns with pros/cons.
5. **Governance routing** — Identify approvers for each scenario branch.
6. **Risk register** — Quality, security, business continuity, comms.
7. **Publish playbook** — Single link/summary for core team.
8. **Trigger downstream** — Hand to **RFX Creator** and **eRFX Agent** with locked scope.

## Escalation

**Scope creep** from business sponsors → formal change control before re-baselining suppliers.
