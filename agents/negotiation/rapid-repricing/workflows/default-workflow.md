# Default workflow — Rapid Repricing

## Goal

Produce **Finance-reviewable** scenario math during a time-boxed negotiation.

## Steps

1. **Lock baseline** — Price list version, date, currency basis.
2. **Capture shocks** — Volume, FX, index, surcharge toggles.
3. **Compute scenarios** — Deterministic preferred; LLM explains.
4. **Validate non-linearities** — Tier breaks, mins/maxs, brackets.
5. **Cross-check** — Spot-check 2–3 SKUs manually if automated.
6. **Package** — Table + assumption log + open questions.
7. **Approval** — Deal desk sign-off for out-of-policy deviations.

## Escalation

Conflicting tax treatment → pause; involve Finance/tax.
