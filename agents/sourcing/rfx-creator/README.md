# RFX Creator

**RFX Creator** focuses on **drafting and assembling** RFI, RFP, and RFQ content—turning scope conversations into **structured documents**, **scoring models**, and **review-ready outlines**.

## Why it matters

Weak requirements create weak markets. This agent helps teams produce **complete, evaluable** RFx packages faster—without skipping security, data, or commercial guardrails.

## Common inputs

- Technical scope narratives, SLAs, data handling needs
- Commercial goals: TCO elements, incumbents, transition assistance
- Clause library excerpts **approved** for use

## Typical outputs

- Section-by-section RFx outline with named owners
- Weighted scoring rubric draft aligned to strategy
- Red-flag checklist (missing insurance reqs, unclear IP, etc.)

## Workflow in practice

Runs **before publication**; Legal reviews binding terms. Hands structured artifacts to **eRFX Agent** for event operations.

## Extension opportunities

- Auto-map requirements to **security questionnaires**
- Generate **lot-specific** pricing worksheets from templates

## Differentiation

- **eRFX Agent** runs the **event process**.
- **RFX Creator** builds the **RFx product** itself.
