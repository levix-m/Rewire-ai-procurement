# Default workflow — Supplier Performance Cockpit

## Goal

Produce a **trusted monitoring narrative** ahead of business reviews.

## Steps

1. **Refresh KPIs** — confirm as-of timestamp and completeness.
2. **Baseline comparisons** — prior period, same period last year, target.
3. **Anomaly detection** — rules + stats; de-duplicate correlated signals.
4. **Context enrichment** — major known events (strikes, launches) if provided.
5. **Draft storyline** — concise, layered (exec vs operational detail).
6. **Export spec** — fields, filters, refresh cadence.
7. **Hand-off** — if systemic failure suspected, route to **SPM** with evidence bundle.

## Escalation

KPI definitions disputed → pause narrative; escalate to data owner.
