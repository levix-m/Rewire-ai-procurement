# Default workflow — eRFX Agent

## Goal

Close an event with **complete, compliant submissions** and a **clean evaluation start**.

## Steps

1. **Kickoff** — Confirm charter, supplier contacts, confidentiality acknowledgments.
2. **Publish** — RFx artifacts versioned; hash or record stored.
3. **Supplier enablement** — webinar/office hours as needed.
4. **Q&A cadence** — intake, triage, legal review, publish responses.
5. **Amendment control** — track scope changes; extend deadlines fairly if material.
6. **Submission gate** — validate files; virus scan; completeness checks.
7. **Integrity review** — conflicts, declinations, odd pricing patterns flagged (process only).
8. **Hand-off** — indexed bundle + clarifications log to evaluators.

## Escalation

**Material fairness issue** (e.g., selective information) → pause event; escalate to governance.
